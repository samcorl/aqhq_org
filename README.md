# ArtQuest HQ

Mobile-first linklist app for ArtQuest at Santa Rosa High School.

**Live Site:** https://www.aqhq.org

## Features
- Mobile-first responsive design
- 8 Visual & Performing Arts specialties
- Contact information and social media links
- Orange & black school colors theme

## Tech Stack
- Static HTML/CSS
- Font Awesome icons
- Nginx hosting
